This folder contains auxiliary code for my paper.

- anira:
  This represents the state of the anira repository at the time of me writing the paper. Newer versions have been released since. Additionally, I had to pin a specific version of the onnxruntime-web package.
- anira-web-benchmarks:
  This contains the benchmarking suite.
- anira-web-r:
  This contains the code used for analysing the benchmark results, as well as the results of the benchmarks as input, and the results of the analysis as the output.
